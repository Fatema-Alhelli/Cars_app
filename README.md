# 🚗 Car Price Prediction App

This project is part of Machine Learning Exercise 3. It utilizes a trained machine learning model to estimate car prices based on user-inputted specifications and provides real-time predictions via an interactive web dashboard.

## 📌 Features
- **Interactive UI:** Built using Streamlit for a simple and intuitive user experience.
- **Machine Learning Pipeline:** Utilizes a pre-trained model (`model.pkl`) to make accurate price predictions.
- **Real-Time Prediction:** Instant calculation upon clicking the prediction button.

## 🛠️ Tech Stack
- **Python**
- **Streamlit** (Deployment & Frontend)
- **Scikit-Learn / Joblib** (Model Inference)
- **Pandas & NumPy** (Data Processing)
